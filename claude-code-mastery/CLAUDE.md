# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## 언어 및 커뮤니케이션 규칙

- **기본 응답 언어**: 한국어
- **코드 주석**: 한국어로 작성
- **커밋 메시지**: 한국어로 작성
- **문서화**: 한국어로 작성
- **변수명/함수명**: 영어(코드 표준 준수)

## 프로젝트 현황

현재 이 저장소는 비어 있는 상태로, 아직 소스 코드·설정 파일·의존성 정의가 없습니다.
아래 섹션들은 코드가 추가되는 시점에 실제 내용으로 채워야 합니다.

- **빌드 / 테스트 / 린트 명령**: (코드 및 빌드 도구 추가 후 작성 — 예: 전체 테스트 실행, 단일 테스트 실행 명령 포함)
- **아키텍처 개요**: (여러 파일에 걸친 큰 그림 구조가 생기면 작성)

> 새 코드를 추가하거나 빌드 도구를 도입한 뒤에는 `/init`을 다시 실행하거나
> 위 섹션을 직접 갱신해, 이후 Claude Code 인스턴스가 빠르게 작업할 수 있도록 유지하세요.
